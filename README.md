# Chatbot_aiCodeSmith AI is an intelligent, interactive chatbot designed to assist users with programming and theoretical queries. Built using Python and Streamlit, and powered by the Groq LLM API, it provides answers in multiple formats:

Code Only – Returns executable code snippets for programming questions.

Theory Only – Provides detailed explanations for theoretical questions.

Code + Theory – Combines both explanations and code for comprehensive understanding.

Key Features

Interactive Chat Interface: Users can type questions and receive instant AI-generated responses.

Copy Functionality: Easily copy code or theory directly from the app.

Chat History: Sidebar keeps track of all previous queries and answers.

Cross-Device Access: Can be deployed on Streamlit Cloud for access from any PC or mobile browser without exposing the code.

Secure API Key Handling: API keys are securely managed using Streamlit secrets.

Technology Stack

Frontend: Streamlit

Backend / AI: Groq LLM API (Llama 3.1)

Programming Language: Python

Deployment: Streamlit Cloud / Optional local network deployment

Use Case:
CodeSmith AI is ideal for students, developers, and hobbyists who want instant AI assistance for coding tasks, programming theory, and learning purposes.
